# Coffee Shop - Bar

Proyecto final de e-commerce para el curso de Desarrollo Web.

## Descripcion

Sitio web interactivo para una cafeteria-bar. La aplicacion consume productos desde una API REST, renderiza tarjetas dinamicamente y permite agregar productos a un carrito de compras persistente con localStorage.

## Funcionalidades

- Consumo de API REST con Fetch y async/await.
- Renderizado dinamico de productos en cards.
- Carrito de compras con localStorage.
- Contador dinamico de productos.
- Edicion de cantidades, eliminacion de productos y vaciado del carrito.
- Calculo de subtotal y total.
- Simulacion de compra con boton de checkout.
- Formulario de contacto funcional con Formspree.
- Diseno responsive con CSS externo, Flexbox, Grid, Bootstrap y media queries.
- Buenas practicas de accesibilidad y SEO.

## Tecnologias utilizadas

- HTML5 semantico
- CSS3
- Bootstrap 5
- Flexbox
- CSS Grid
- JavaScript
- Fetch API
- localStorage
- Formspree
- Font Awesome
- Google Fonts

## Instalacion y uso

1. Descargar o clonar el repositorio.
2. Abrir `index.html` en el navegador.
3. Navegar los productos, agregar al carrito y probar la simulacion de compra.
